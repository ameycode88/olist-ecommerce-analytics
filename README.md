# olist-ecommerce-analytics

# Brazilian Olist E-commerce Dataset:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/suggestions


## Excel Dashboard

Features:

- KPI Dashboard
- Revenue Analysis by Category
- Payment Installment Analysis
- Interactive PivotTables
- Slicers
- Conditional Formatting
- Excel Formulas (SUMIF, COUNTIFS, AVERAGEIFS)
